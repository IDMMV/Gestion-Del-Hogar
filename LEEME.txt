MI HOGAR AL DÍA — PWA V5.8.0

SUBIR A GITHUB PAGES
1. Descomprime este ZIP.
2. Sube index.html, manifest.webmanifest, service-worker.js y la carpeta icons al nivel principal del repositorio.
3. No subas solamente index.html: la PWA necesita todos los archivos.
4. Abre https://idmmv.github.io/Gestion-Del-Hogar/
5. En Chrome Android aparecerá el botón “Instalar aplicación” o la opción “Agregar a pantalla principal”.

IMPORTANTE
- GitHub Pages debe estar publicado mediante HTTPS.
- Después de reemplazar archivos, recarga la web o elimina la versión instalada anterior si conserva caché antigua.
- Supabase Site URL y Redirect URL deben apuntar a https://idmmv.github.io/Gestion-Del-Hogar/
